View Transitions

The View Transitions API provides a mechanism for easily creating animated transitions between different website views. This includes animating between DOM states in a single-page app (SPA), and animating the navigation between documents in a multi-page app (MPA).üï∫üèºüôåüèΩ

  animation-direction: both;
}
